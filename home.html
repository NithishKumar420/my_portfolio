HAB (Hospital System)

A system to book hospital appointments online quickly and easily.

The Hospital Appointment Booking System (HAB) is a web-based application designed to simplify the process of scheduling medical appointments for patients and healthcare providers. This system allows users to easily book, manage, and track appointments online without the need for physical visits or long waiting times. The platform provides a user-friendly interface where patients can register, log in, view available doctors, and book appointments based on their preferred date and time. On the admin side, the system enables efficient management of doctors, patient records, and appointment schedules. This project was developed using modern web technologies to ensure responsiveness, performance, and ease of use. It helps reduce manual workload in hospitals and improves overall patient experience by providing a seamless digital solution. Key Features: User registration and login system Doctor listing and availability management Appointment booking and cancellation Admin dashboard for managing users and schedules Responsive design for mobile and desktop Technologies Used: Frontend: HTML, CSS, JavaScript Backend: Python (Flask) Database: MySQL

EMS (Employee System)

A system to manage employee records and data digitally.

The Employee Management System is a web-based application designed to streamline and automate the management of employee data within an organization. This system helps in efficiently handling employee records, roles, attendance, and other essential details in a centralized platform. The application provides a secure interface where administrators can add, update, view, and delete employee information. It reduces manual paperwork and improves accuracy by digitizing employee-related operations. The system is built with a focus on simplicity, performance, and ease of use. It also supports role-based access, ensuring that only authorized users can manage sensitive employee data. The responsive design allows access across devices, making it convenient for both administrators and staff. Key Features: Employee registration and profile management Add, update, and delete employee records Role-based access control (Admin/User) Search and filter employee details Secure login and authentication system Responsive design for mobile and desktop Technologies Used: Frontend: HTML, CSS, JavaScript Backend: Python (Flask) Database: MySQL

Job Portal

A platform connecting job seekers and employers.

The Job Portal System is a web-based application designed to connect job seekers with employers in a simple and efficient way. This platform allows users to search for jobs, apply online, and manage their applications, while employers can post job openings and find suitable candidates. The system provides a user-friendly interface where job seekers can register, create profiles, upload resumes, and explore job listings based on their skills and preferences. Employers can manage job postings, review applications, and shortlist candidates. This project helps reduce the gap between job seekers and recruiters by providing a centralized digital platform, improving the hiring process and saving time for both parties. Key Features: User registration and login (Job Seeker & Employer) Job posting and management by employers Job search and filtering options Resume upload and application tracking Admin panel for managing users and job listings Responsive design for mobile and desktop Technologies Used: Frontend: HTML, CSS, JavaScript Backend: Python (Flask) Database: MySQL
